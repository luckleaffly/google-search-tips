### 搜索运算符
---
关键字      |说明                |例子 
------------|--------------------|---------
site        |仅从特定网站或网域获得搜索结果                             |site:www.tmall.com 手机
link        |查找链接到某个特定网页的网页(网页中包含www.tmall.com链接) |link:www.tmall.com
related     |查找与该网站相似的网站                                     |related:www.tmall.com
cache       |查看网页在Google上的缓存                                   |cache:www.tmall.com
info        |查询site/link/related/cache四种信息，目前无效              |info:www.tmall.com
inurl       |查找在URL地址里有搜索关键词的页面,bd也支持                 |inurl:ocj
allinurl    |与allintitle: 类似。allinurl:SEO 搜索引擎优化,就相当于 ：inurl:SEO inurl:搜索引擎优化|
intitle     |查找在网页标题里有搜索关键词的页面                         |intitle:手机
allintitle  |搜索返回的是页面标题中包含多组关键词的文件例如 ：allintitle:SEO 搜索引擎优化就相当于：intitle:SEO intitle:搜索引擎优化,返回的是标题中中既包含“SEO”，也包含“搜索引擎优化”的页面|
inanchor    |查找特定关键词的锚文本链接,如<a href="xx">手机</a>         |inanchor:手机
intext      |查找你想要搜索的内容出现在网页正文里面                     |intext:手机
filetype    |查找pdf,xml,xls,txt,doc,csv等特定格式的结果                |filetype:pdf java
~~define~~  |~~快速查询关键词的定义~~                                   |~~define:手机~~
混合运算符  |在站点congbuluo或baidu下面查包含apple的且不包含iphone的网页|site:chongbuluo.com OR site:baidu.com "apple" -iphone
双引号      |把搜索词放在双引号中，代表完全匹配搜索，也就是说搜索结果返回的页面包含双引号中出现的所有的词，连顺序也必须完全匹配。bd和Google 都支持这个指令。|例如搜索： “seo方法图片”
减号        |减号代表搜索不包含减号后面的词的页面。使用这个指令时减号前面必须是空格，减号后面没有空格，紧跟着需要排除的词。Google 和bd都支持这个指令。      |"搜索 -引擎" 返回的则是包含“搜索”这个词，却不包含“引擎”这个词的结果
星号        |星号*是常用的通配符，也可以用在搜索中。百度不支持*号搜索指令。比如在Google 中搜索：搜索*擎其中的*号代表任何文字。返回的结果就不仅包含“搜索引擎”，还包含了“搜索收擎”，“搜索巨擎”等内容。
linkdomain  |linkdomain:指令返回的是某个域名的反向链接,得到外部链接     |linkdomain:http://www.tmall.com -site:http://www.tmall.com
组合用法    |返回的就是url 中包含gov，页面中有“减肥”这个词的页面        |inurl:gov 减肥

### 搜索技巧
关键字      |说明
------------|---------------
网络历史记录                        |网络历史记录可让您查阅访问过的网站的日志、浏览操作的时间轴以及搜索自己的在线历史记录。要试用该功能，请访问 www.google.com/history。
从简单的字词入手                    |无论您要查找什么，我们的宗旨都是“尽量简单”，即从输入基本的名称或字词开始。如果您要查找某个地方的某商家或产品，请输入相应的名称以及城镇名或邮政编码。搜索"烤鸭店"
自动纠正拼写错误                    |无论您的拼写是否正确，Google 的拼写检查功能都会自动默认使用相关字词最常用的拼写形式。搜索“gooogle”
使用适合网络搜索的字词              |搜索引擎的工作原理是将您输入的字词与网络上的网页内容进行匹配。因此，为了获得最佳结果，请使用最有可能出现在网页上的字词。例如，请输入“头痛”，而不是“我的头不舒服”，因为“头痛”才是医疗网站上会使用的词。
字词越少，结果越多                  |一两个简单的搜索字词通常可让您获得最全面的搜索结果。请先用简短的字词开始搜索，然后添加其他字词来优化搜索结果。
完全匹配搜索                        |在字词前后分别加上中文或英文双引号（如"[任意字词]"），可精确搜索字序相同的词组。请注意，使用双引号进行搜索可能会排除一些相关的搜索结果。例如，搜索“Alexander Bell”可能会漏掉涉及 Alexander G. Bell 的网页。搜索“"漫卷诗书喜欲狂"”
使用描述性字词                      |字词越独特，您越有可能获得相关程度较高的结果。因此，[名人铃声] 的搜索效果可能要优于 [名人声音]。不过请注意，如果您输入的搜索字词并不常用，那么即使它的意思正确，也可能与您所需的网页不匹配。
无需担心大小写问题                  |搜索字词不区分大小写。搜索“qq”与“QQ”所得到的结果是一样的。
在特定网站内搜索                    |如果您希望在特定的网站或特定类型（.org 或 .edu）的网站中进行搜索，请在查询内容前加上 site:，例如：“site:edu”或“site:nytimes.com”。搜索“op-ed site:nytimes.com”
无需担心标点符号问题                |搜索字词会忽略标点符号，其中包括 @#%^*()=[]\ 和其他特殊字符。
按文件类型搜索                      |要搜索特定类型的文件（例如，PDF、PPT 或 XLS），您可以添加 filetype: 和文件格式缩写（以 3 个字母表示）。搜索“医疗保险法 filetype:ppt”
包括或忽略您搜索查询中的字词和字符  |如果的、& 等常见字词字符对您的搜索十分重要（例如在电影或图书名称中），那么您可以在其前后分别加上双引号（即“的”）加以强调。您还可以使用减号 (-) 来指定不希望结果中包含的特定字词，例如食谱中的某些原料。“火锅食谱 - 香菜”
查找相关网页                        |您可以使用 related: 运算符来查找具有类似内容的网页，方法是输入“related:”，然后加上相关网址。例如，如果您找到了一个喜欢的网站，可以尝试使用“related:[输入网址]”来查找类似的网站。“related:www.meishichina.com/eat/caipu/”
查找食谱                            |搜索特定的食谱或原料后，只需选择左侧面板中的“Recipes”（请注意，中文版的左侧面板中尚没有对应的“食谱”选项），即可看到相应的食谱搜索结果。搜索“东坡肘子”
搜索某个范围内的数字                |要保证不超出预算，您可以在数额间加入“..”字符串，从而只搜索某个数值范围内的商品。小米电视 ¥1000..¥2000
进行数值换算                        |输入数字和度量单位，即可进行任意度量单位换算，例如英里换算为公里或盎司换算为公升。搜索“1 英里等于多少公里”
查询时间                            |要查看世界各地的当前时间，只需搜索“时间”和城市名或国家/地区名即可。搜索“
进行货币换算                        |搜索“[币种 1] 兑换 [币种 2]”，即可了解当前汇率。搜索 “500 美元 = ? 人民币”
查看天气                            |只需搜索“天气”和相应的邮政编码或全球任意城市名称，即可查询当地的当前天气和天气预报。单独输入“天气”，即可获取您当前所在地的天气报告。
查询实时股票价格                    |只需在搜索框中输入任意股票代码，即可在搜索结果中获取即时更新的实时股票报价。点击搜索结果页上的相应链接，即可查看由 Google 财经提供的详细市场分析。
查询字词定义                        |在任意字词前加上 define:，即可查询其定义。搜索“define: loquaciousness”
使用高级图片搜索功能进行搜索        |使用高级图片搜索功能，可查找某确切尺寸、颜色或类型的照片或绘画。借助左侧面板中的各种工具，您可以对搜索结果进行过滤，将结果限制为仅包含人物照片、剪贴画、高分辨率图片或可用于商业用途的图片。搜索“三清山”
数学计算                            |您只需在搜索框中输入任意数学公式，我们就会帮您计算出相应的结果。搜索“100*3.14-cos(83)=”
按地点搜索                          |在任何地点查询（如“烧烤店”）后添加邮政编码，即可获取最近的几家烧烤店及其电话号码、地图，甚至还有评论。如果您未加上邮政编码，那么我们会推荐您附近的地方。搜索“barbecue ribs 35201”（烧烤店 35201）
按邮编和区号搜索地点                |想知道某个邮政编码或区号对应的是哪个地点吗？只需将其输入搜索框即可。100081
网上购物，货比三家                  |使用搜索结果页左侧面板上的“购物”链接，即可在网络上购买商品并比较价格。计算机
查询电影放映时间                    |搜索电影名称或只搜索“电影”，即可查看您所在地区的影院位置和电影放映时间。
用图书搜索看免费书                  |选择搜索结果页左侧面板中的“图书”，即可免费全文阅读公共领域内的图书，如《红楼梦》。红楼梦
限定数字范围                        |要限定某个数字范围，请先输入数字范围，然后输入“..”和相应的单位。例如，如果您要搜索马力超过 300 的汽车，请搜索“汽车 300..马力”。一些其他示例如下：220..伏、1..转或者 8000.. 毫安电池
查看航班时刻表                      |要查看飞往特定地点或从该处起飞的航班时刻表，请输入“航班飞离”或“航班飞往”和您感兴趣的城市或机场的名称。您还可以使用“出发地”或“目的地”添加其他地点，并查看特定路线的时刻表。*美国航班时刻表数据由 OAG 和 Innovata 的 QuickTrip 提供。搜索“flights from chicago to denver”（芝加哥飞往丹佛的航班）
检查健康状况                        |要了解某个常见疾病或病症，请在搜索框中输入相应字词，系统将返回相关专业文章的开头内容。点击该搜索结果，可以阅读整篇文章。哮喘

看不同视角的国际新闻                |搜索任意主题，然后点击左侧面板中的“新闻”标签，即可获取来自世界各地的新闻搜索结果。奥巴马
搜索地址                            |只需输入相应的城市名或邮政编码和 [地图] 一词，即可查看该地点的地图。点击该地图，即可在 Google 地图中查看相关详情。搜索“200051 地图”
类似字词                            |在搜索字词前加上 ~ 符号，即可获取包含其同义词的搜索结果。例如，搜索英文“Christmas ~dessert recipes”（圣诞~甜点食谱），将会获得包含甜点以及糖果、饼干和其他零食的搜索结果。
